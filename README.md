# Data Warehouse and Analytics Project
Welcome to the Data Warehouse and Analytics Project Repository!
This project demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse to generating actionable insights. Designed as a portfolio project highlights the industry best practices in data engineering and analytics.

-----
**Project requirements**

**##Building the data warehouse**(Data engineering)

## Objective
Develop a modern data warehouse to consolidate sales data using SQL server, enabling analytics so that the stakeholders can take informed decisions through reporting.

### Specifications
-Data Sources: Import data from two source systems provided(crm and erp as csv files)
-Data Quality: Clean and resolve the data quality issue before analysis
-Intergration: Combine both sources into single, user friendly data models designed for analysis
-Scope: Focus only on the latest data, historization is not required.
-Documentation: Provide clear documentation of data model supporting both business 
 stakeholders and the analytics team.

 -----

 ### BI : Analytics and reporting (Data Analytics)

 ## Objective
 Develop SQL based analytics to deliver the insights of
 1. Customer Behviour
 2. Product Performance
 3. Sales Trends

    These insights empower the stake holders to take the right decision.
